Local variables have no shared access by multiple threads, so it's meaningless to make it as an lock.

    
    public void run()
    {
        Object lock=new Object();
        while( true )
        {
            try
            {
                synchronized( lock )
                {
                   ...//business logic
                }
            }
            catch( Exception e )
            {
                LOGGER.error( "Error found:", e );
            }
        }
    }

The call on synchronized( lock ) dose not make any sense, since lock is impossible to be hold by other thread
